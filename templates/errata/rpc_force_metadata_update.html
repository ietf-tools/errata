Force RFC Metadata Update

⚠ WARNING: This operation can be expensive and should only be used when necessary!

This page allows you to trigger an update of RFC metadata for all RFCs or a specified list of RFC numbers. It should only be used when an error has occured with the automated synchronization with the datatracker. If possible sync only a small number of affected RFCs. Syncing the entire series is an expensive operation that should not be run frequently.
